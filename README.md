## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SpaceDesigns/Atelier.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build the client
   ```sh
   npm run build
   ```
4. Run the server
   ```sh
   npm run server-dev
   ```
